# AI Boundary

## Principle

AI assists communication and authoring. It does not own computational truth.

## AI may not create or modify

- semantic nodes;
- source ranges;
- graph relations;
- trace events;
- binding snapshots;
- deterministic pattern hits;
- scene layout;
- scene actions;
- verification results.

## AI may do

- simplify verified explanations;
- translate;
- propose analogies;
- draft practice variations;
- create distractors;
- suggest transfer problems;
- summarize imported projects;
- propose candidate patterns;
- adapt vocabulary;
- help authors draft lessons.

## Grounding package

Each explanation receives only the relevant source excerpt, selected nodes, relations, trace steps, patterns, allowed vocabulary, learner level, and unsupported regions.

## Status labels

AI-generated items carry `candidate`, `reviewed`, or `approved`.

## Publishing

No AI-generated content is published without human verification.

## Failure behavior

If AI is unavailable, the deterministic Code Lens remains fully functional.
