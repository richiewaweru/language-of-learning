# Research and Learning Validation

## Hypothesis

Structural-first synchronized instruction helps novices form reusable schemas faster than conventional prose-and-code explanation.

## First experiment

Topic: accumulator pattern.

Control:

- definition;
- code example;
- prose explanation;
- equal practice time.

Lens:

- canonical shape;
- synchronized code mapping;
- controlled trace;
- state transitions;
- pattern explanation;
- equal practice time.

## Measures

- time to benchmark;
- output prediction;
- structural identification;
- debugging;
- reconstruction;
- transfer;
- delayed retention;
- cognitive effort;
- English proficiency as moderator.

## Guardrails

- no fixed learning-style labels;
- satisfaction is not a learning outcome;
- animation alone is not the treatment;
- time-on-task must be controlled;
- transfer uses unseen surface details;
- null results are reported honestly.

## Initial success signal

Lens learners solve unseen accumulator examples more accurately or faster without more instructional time.

## Kill criterion

If two iterations show no transfer or efficiency advantage, revise the strong claim.
