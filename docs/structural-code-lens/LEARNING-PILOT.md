# Structural Code Lens learning pilot worksheet

Use anonymous participant codes. Do not record names, email addresses, source code belonging to an employer, or other identifying data.

For each participant and structurally similar unseen example, record:

| Participant code | Example | Next-step prediction correct | Explanation correct | Seconds to locate current state | Confusion observed | Transfer task correct |
|---|---|---:|---:|---:|---|---:|
| | | | | | | |

Protocol:

1. Show Flow without Guided Trace and ask for the next execution event.
2. Ask the participant to explain the current item, operation, state transition, and return status.
3. Switch to State and time how long it takes to identify the latest changed value.
4. Allow Guided Trace, note any symbol or causality confusion, then present an unseen structurally similar program.
5. Report aggregate accuracy, median state-location time, confusion themes, and transfer accuracy. Keep this artifact marked pending until a real pilot is completed.
