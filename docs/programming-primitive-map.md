# Programming Primitive Map — Teaching Atlas (Reference Only)

**Not parser ontology.** The parser emits only the 12 node kinds in `semantic-contract.md`. This atlas guides what the system teaches and links to lessons.

## Four levels

### Semantic primitives (parser-aligned)
Value, binding, collection, operation, sequence, branch, loop, function, call, return, mutation, effect.

Binding roles: `constant`, `parameter`, `local`, `iterator`, `state`.

### Language constructs (broader)
Variable, parameter, argument, expression, conditional, loop, list, object, class, method, module, exception.

### Recurring patterns (v0.1 registry)
| Pattern | Learner verb |
|---|---|
| ACCUMULATE | build up one value |
| COUNT | count matching items |
| FILTER | keep only some items |
| TRANSFORM | change every item |
| SEARCH | stop at the first match |
| GUARD | reject bad input first |

### Algorithms and structures (future)
Traversal, two pointers, sliding window, divide and conquer, recursion tree, graph search, event-handler-state-render.

## Admission test

A concept enters the atlas when it: (1) represents a distinct mental operation; (2) cannot be reduced without losing value; (3) recurs across examples; (4) supports a stable visual law; (5) improves transfer; (6) has fixtures and checks.
