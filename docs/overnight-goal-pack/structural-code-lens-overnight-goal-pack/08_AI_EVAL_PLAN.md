# AI Evaluation v0

## Cases
Accumulation, count, linear search, mutation, binary search, generic supported, unsupported.

## Questions
- What happened?
- Why did the value change?
- Where did the current value come from?
- What happens next?
- Explain the whole process.
- Explain it more simply.

## Targets
- At least 95% factual agreement.
- Zero invented return values.
- Zero invented branch outcomes.
- 100% unsupported honesty.
- At least 90% correct semantic vocabulary.
- At least 99% schema validity after one retry.
- Fallback available for every provider failure.

Record provider, model, latency, usage, retries, fallback, schema status, grounding, and clarity notes.
